# That1Circle
No bells. No Whistles.
Tap on the circles to make them go away. 
Whatever you do, however many fingers/hands you use 
Don’t tap outside the circles.
Simple, right? 
Well, that bar up top is shrinking… 
How far will you go? 
How long will you last?
Oh, That1Circle...
